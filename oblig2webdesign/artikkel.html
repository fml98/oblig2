Git/Github

Github er en fantastisk nettside for begavede mennesker som vet hva de driver med. Dette er desverre ikke tilfellet for meg.

Når man tar en nærmere titt på Github finner man fort ut at det er en del kule instillinger man kan ta i bruk. Github sin funksjonalitet er hovedsakelig en skylagringstjeneste hvor flere kan jobbe på samme fil å redigere, uten problem. Den lagrer i tillegg eldre verisjoner av github, som gjør at det blir enkelt og strukturere arbeidet på en effektiv og brukervennlig måte.

Du kan sammenligne Github med word online, bare med bedre funksjonalitet, og flere muligheter, og istaden for vanlig tekst, så lagrer den koder.
